# nalclip

날클립 백엔드/뷰 프론트엔드, typescript + next.js + prisma + postgresql + docker-compose 그리고 semantic ui를 곁들인

클립보드 이벤트를 [윈도우 클라이언트](https://github.com/20chan/nalclip.client)에서 후킹해서 쏴주면 서버에서 기억해준다
